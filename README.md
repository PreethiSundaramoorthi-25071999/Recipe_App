1. This repository is about CRUD operation.
2. Download the Express, nodemon, mongoose and dotenv packages.
3. Create the separate folders for mongoose, routes and controller. In each folder, create the required files.
4. Connect all the folder path in server.js file
5. In routes file, create the router for create, read, update and delete operations.
6. In controller file, write the required code for create, read, update and delete operations.
7. In Postman, we can create the separate recipe for each dishes.
8. Once we done with the creation, the data will be automatically store in the MongoDB.
9. We can do the CRUD operations for each data by calling each id number of the data.
